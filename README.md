# Movie Recommendation Engine

A production-grade Movie Recommendation System built using **Surprise** and **Scikit-learn**, implementing multiple recommendation strategies including:

- User-Based Collaborative Filtering
- Matrix Factorization (SVD)
- Content-Based Filtering
- Cold-Start Recommendations

The project is fully containerized using **Docker** and exposes recommendations via a **FastAPI** service.

---

## Project Goals
- Build an end-to-end recommendation engine similar to Netflix/Amazon
- Implement and compare multiple recommendation algorithms
- Evaluate models using industry-standard metrics
- Serve recommendations through a REST API
- Ensure reproducibility with Docker
